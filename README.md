# ⚡ ORION — AI Marketing OS

> Go from marketing goal → strategy → content → distribution → analytics in minutes.

ORION is a full-stack SaaS platform that acts as an AI-powered Marketing Operating System. Six specialized AI agents collaborate to plan, create, distribute, and optimize marketing campaigns across all major channels.

## Getting Started

### Prerequisites

- Node.js 20+
- Docker + Docker Compose
- An Anthropic API key

### 1. Clone and set up

```bash
git clone https://github.com/your-org/orion.git
cd orion

# One-command setup (installs deps, starts Docker, migrates DB, seeds fixtures)
./infra/scripts/dev-setup.sh
```

### 2. Configure environment

```bash
cp .env.example .env.local
# Edit .env.local — at minimum you need:
#   DATABASE_URL (pre-filled for local Docker)
#   ANTHROPIC_API_KEY
#   NEXTAUTH_SECRET (generate: openssl rand -base64 32)
```

### 3. Start development

```bash
npm run dev
# Starts both Next.js (port 3000) and Express API (port 3001) in parallel
```

**Dev URLs:**
- App: http://localhost:3000
- API: http://localhost:3001/health
- Inngest dashboard: http://localhost:8288
- Email (MailHog): http://localhost:8025
- DB Studio: `npm run db:studio`

**Dev login:** `dev@acme.com` / `password123`

## AI Agent System

All AI calls are **server-side only**. The `ANTHROPIC_API_KEY` is never exposed to the browser.

```
Browser → Next.js API Route → Express API → @orion/agents → Anthropic API
                                                ↓
                                         (streaming SSE back to browser)
```

### Agents

| Agent | Class | Trigger | Output |
|---|---|---|---|
| Strategist | `MarketingStrategistAgent` | Goal created | Full 30-day marketing plan |
| Content Creator | `ContentCreatorAgent` | Channel selected | Platform-native copy (streaming) |
| Optimizer | `OptimizationAgent` | Manual trigger | Data-backed recommendations |
| Distribution | *(Phase 2)* | Schedule reached | Published post |
| Analytics | *(Phase 2)* | Inngest cron | Event rollups |
| CRM Intelligence | *(Phase 2)* | Contact event | Lead score update |

## Security Notes

- `ANTHROPIC_API_KEY` is server-side only — enforced by `BaseAgent` constructor check
- OAuth tokens stored AES-256 encrypted in `channel_connections.access_token_enc`
- All DB queries use Drizzle ORM (parameterized — no SQL injection)
- Row-level security: every query is scoped by `org_id` from the validated session
- Stripe webhooks validated by signature before processing
- Rate limiting on all AI generation endpoints (10 req/min per user)
- CSP headers configured in `next.config.js`
